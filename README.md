# NxVpn

Android VPN client with support for multiple protocols.

## Download

Go to [Releases](../../releases) to download the latest APK.

## Features

- VPN / Proxy
- SSH Tunnel
- V2Ray / Xray

## Requirements

- Android 5.0 (API 21) or higher
